# MySQL

## Mode ligne de commande 

```$ mysql -u root -p```

## Changer de mot de passe

```$mysqladmin -u root -p password NEWPASSWORD```

## Lister les bases de données

```$ SHOW DATABASES
```

## Changer de base de données

```
$ use <dbname>
```
## Lister les tables

```
$ SHOW TABLES
```

## Créer une table

```
$ CREATE DATABASE dbname;
```
