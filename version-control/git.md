# GIT
## Création
```$git clone ssh://user@domain.com/repo.git```
```$git init```

## Changements en local
$ git status
$ git diff
$ git add .
$ git add -p <file>

## Configurer un utilisateur
### Conventionnel
git config —global user.email “you@example.com"
git config —global user.name “Your name"

### Via fichier de config
